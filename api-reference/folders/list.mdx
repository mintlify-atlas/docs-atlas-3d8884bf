---
title: "List Folders"
api: "GET /api/folders"
description: "Retrieve all folders for the authenticated user"
---

## Query Parameters

<ParamField query="includeArchived" type="boolean" default="false">
  Include archived folders in the response
</ParamField>

## Response

Returns an array of folder objects.

<ResponseField name="folderId" type="number">
  Unique identifier for the folder
</ResponseField>

<ResponseField name="userId" type="number">
  ID of the user who owns the folder
</ResponseField>

<ResponseField name="tags" type="array">
  Array of tag objects associated with the folder
</ResponseField>

<ResponseField name="name" type="string">
  The name of the folder
</ResponseField>

<ResponseField name="description" type="string">
  Description of the folder
</ResponseField>

<ResponseField name="color" type="string">
  Color code for the folder
</ResponseField>

<ResponseField name="icon" type="string">
  Icon identifier for the folder
</ResponseField>

<ResponseField name="parentFolderId" type="number">
  ID of the parent folder
</ResponseField>

<ResponseField name="displayOrder" type="number">
  Display order of the folder
</ResponseField>

<ResponseField name="isArchived" type="boolean">
  Whether the folder is archived
</ResponseField>

<ResponseField name="createdAt" type="string">
  ISO 8601 timestamp of folder creation
</ResponseField>

<ResponseField name="updatedAt" type="string">
  ISO 8601 timestamp of last update
</ResponseField>

<RequestExample>
```bash cURL
curl -X GET https://api.taskpro.com/api/folders \
  -H "Authorization: Bearer YOUR_TOKEN"
```

```bash cURL (with archived)
curl -X GET "https://api.taskpro.com/api/folders?includeArchived=true" \
  -H "Authorization: Bearer YOUR_TOKEN"
```
</RequestExample>
