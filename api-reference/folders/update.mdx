---
title: "Update Folder"
api: "PUT /api/folders/{folderId}"
description: "Update an existing folder's properties"
---

## Path Parameters

<ParamField path="folderId" type="number" required>
  The unique identifier of the folder to update
</ParamField>

## Request

All fields are optional. Only include the fields you want to update.

<ParamField body="name" type="string">
  The name of the folder (max 100 characters)
</ParamField>

<ParamField body="description" type="string">
  Description of the folder
</ParamField>

<ParamField body="color" type="string">
  Color code for the folder (e.g., "#FF5733", max 7 characters)
</ParamField>

<ParamField body="icon" type="string">
  Icon identifier for the folder (max 50 characters)
</ParamField>

<ParamField body="displayOrder" type="number">
  Order in which the folder should be displayed
</ParamField>

<ParamField body="isArchived" type="boolean">
  Whether the folder should be archived
</ParamField>

## Response

<ResponseField name="folderId" type="number">
  Unique identifier for the folder
</ResponseField>

<ResponseField name="userId" type="number">
  ID of the user who owns the folder
</ResponseField>

<ResponseField name="tags" type="array">
  Array of tag objects associated with the folder
</ResponseField>

<ResponseField name="name" type="string">
  The name of the folder
</ResponseField>

<ResponseField name="description" type="string">
  Description of the folder
</ResponseField>

<ResponseField name="color" type="string">
  Color code for the folder
</ResponseField>

<ResponseField name="icon" type="string">
  Icon identifier for the folder
</ResponseField>

<ResponseField name="parentFolderId" type="number">
  ID of the parent folder
</ResponseField>

<ResponseField name="displayOrder" type="number">
  Display order of the folder
</ResponseField>

<ResponseField name="isArchived" type="boolean">
  Whether the folder is archived
</ResponseField>

<ResponseField name="createdAt" type="string">
  ISO 8601 timestamp of folder creation
</ResponseField>

<ResponseField name="updatedAt" type="string">
  ISO 8601 timestamp of last update
</ResponseField>

<RequestExample>
```bash cURL
curl -X PUT https://api.taskpro.com/api/folders/42 \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "Work Projects (Updated)",
    "color": "#FF5733",
    "displayOrder": 5
  }'
```

```bash cURL (archive folder)
curl -X PUT https://api.taskpro.com/api/folders/42 \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "isArchived": true
  }'
```
</RequestExample>
