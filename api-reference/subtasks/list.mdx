---
title: "List Subtasks"
api: "GET /api/tasks/{taskId}/subtasks"
description: "Retrieve all subtasks for a specific task"
---

## Request

Retrieves all subtasks associated with a parent task. Only returns subtasks if the authenticated user has access to the parent task.

### Headers

<ParamField header="Authorization" type="string" required>
  Bearer token for authentication
</ParamField>

### Path Parameters

<ParamField path="taskId" type="long" required>
  ID of the parent task
</ParamField>

## Response

Returns an array of subtask objects ordered by their display order.

<ResponseField name="subtaskId" type="long">
  Unique identifier for the subtask
</ResponseField>

<ResponseField name="taskId" type="long">
  ID of the parent task
</ResponseField>

<ResponseField name="title" type="string">
  Title of the subtask
</ResponseField>

<ResponseField name="description" type="string">
  Description of the subtask
</ResponseField>

<ResponseField name="tags" type="array">
  Array of tag objects associated with the subtask
  <ResponseField name="tagId" type="long">
    Tag identifier
  </ResponseField>
  <ResponseField name="name" type="string">
    Tag name
  </ResponseField>
  <ResponseField name="color" type="string">
    Tag color
  </ResponseField>
</ResponseField>

<ResponseField name="durationTagId" type="integer">
  ID of the duration tag
</ResponseField>

<ResponseField name="status" type="string">
  Current status of the subtask
</ResponseField>

<ResponseField name="completedAt" type="string">
  ISO 8601 timestamp when the subtask was completed
</ResponseField>

<ResponseField name="displayOrder" type="integer">
  Display order of the subtask
</ResponseField>

<ResponseField name="createdAt" type="string">
  ISO 8601 timestamp when the subtask was created
</ResponseField>

<ResponseField name="updatedAt" type="string">
  ISO 8601 timestamp when the subtask was last updated
</ResponseField>

<RequestExample>

```bash cURL
curl -X GET https://api.taskpro.com/api/tasks/123/subtasks \
  -H "Authorization: Bearer YOUR_TOKEN"
```

```javascript JavaScript
const response = await fetch('https://api.taskpro.com/api/tasks/123/subtasks', {
  method: 'GET',
  headers: {
    'Authorization': 'Bearer YOUR_TOKEN'
  }
});

const subtasks = await response.json();
```

```python Python
import requests

response = requests.get(
    'https://api.taskpro.com/api/tasks/123/subtasks',
    headers={
        'Authorization': 'Bearer YOUR_TOKEN'
    }
)

subtasks = response.json()
```

</RequestExample>
