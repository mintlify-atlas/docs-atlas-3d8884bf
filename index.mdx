## Welcome to Task Pro API

Task Pro API is a production-ready REST API that provides comprehensive task management capabilities. Built with Spring Boot and Kotlin, it offers a robust backend solution for organizing tasks, managing projects, and tracking progress.

## Technology Stack

Task Pro API is built with modern, production-proven technologies:

- **Spring Boot 4.0** - Enterprise-grade Java framework
- **Kotlin** - Modern, expressive JVM language
- **PostgreSQL** - Reliable relational database
- **JPA/Hibernate** - Object-relational mapping
- **JWT (Auth0)** - Secure token-based authentication
- **Docker** - Containerized deployment support

## Quick Example

Here's a simple example of creating a task:

```bash
curl -X POST https://your-api.com/api/tasks \
  -H "Authorization: Bearer YOUR_JWT_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "title": "Complete project documentation",
    "description": "Write comprehensive API documentation",
    "priority": 1,
    "status": "pending",
    "dueDate": "2026-03-20"
  }'
```
